# `xmath` -- `math` eXtensions

[![GoDoc](https://godoc.org/github.com/go-ng/xmath?status.svg)](https://pkg.go.dev/github.com/go-ng/xmath?tab=doc)

## Quick start

```go
package main

import (
    "fmt"

    "github.com/go-ng/xmath"
)

func main() {
    fmt.Println(xmath.Min(3, 1, 5)) // will print "1"
}
```
